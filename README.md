# Embedded Systems

A collection of Embedded Systems projects built using **ESP32** and **MicroPython**, covering GPIO programming, PWM, stepper motor control, Wi-Fi, IoT, and MQTT. Each project includes source code, circuit connections, and project descriptions.

 📚 Topics Covered
- ESP32
- MicroPython
- GPIO Programming
- PWM
- Stepper Motor Control
- Wi-Fi
- IoT
- MQTT

 📁 Repository Structure
Each project contains:
- Source Code
- Circuit Connections
- Project Description

Projects will be added regularly as I continue learning and building embedded systems applications.
