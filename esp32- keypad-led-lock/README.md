# ESP32 Keypad Password-Protected LED Control

## Objective

Develop a password-protected LED control system using an ESP32, a 4×4 matrix keypad, and MicroPython.

## Components

* ESP32
* 4×4 Matrix Keypad
* LED
* Jumper Wires

## Topics Covered

* ESP32
* MicroPython
* GPIO Programming
* Matrix Keypad Interfacing
* Password Authentication
* Digital Output Control

## Files

* main.py
* diagram.json
* circuit.png

## Description

This project demonstrates a password-protected LED control system using an ESP32 and a 4×4 matrix keypad. Users enter a predefined password through the keypad, and the ESP32 verifies the input. If the password is correct, the LED turns ON; otherwise, access is denied. The project demonstrates keypad interfacing, GPIO programming, password authentication, and embedded system security concepts using MicroPython.
