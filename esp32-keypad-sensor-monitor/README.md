# ESP32 Keypad-Based Sensor Monitoring System

## Objective

Develop a menu-driven sensor monitoring system using an ESP32, a 4×4 matrix keypad, a 16×2 I2C LCD, and MicroPython.

## Components

* ESP32
* 4×4 Matrix Keypad
* 16×2 I2C LCD
* DHT22 Temperature and Humidity Sensor
* Jumper Wires

## Topics Covered

* ESP32
* MicroPython
* GPIO Programming
* Matrix Keypad Interfacing
* I2C LCD Interfacing
* DHT22 Sensor Interfacing
* Menu-Driven User Interface

## Files

* main.py
* diagram.json
* circuit.png

## Description

This project demonstrates a menu-driven sensor monitoring system using an ESP32. A 4×4 matrix keypad allows users to select different sensor readings, which are displayed on a 16×2 I2C LCD. The system provides an interactive interface for monitoring environmental parameters such as temperature and humidity. It demonstrates keypad interfacing, LCD communication, sensor integration, GPIO programming, and embedded systems development using MicroPython.
