# ESP32 LED Brightness Control using Potentiometer

## Objective

Develop an LED brightness control system using an ESP32, a potentiometer, and MicroPython by utilizing Analog-to-Digital Conversion (ADC) and Pulse Width Modulation (PWM).

## Components

* ESP32
* Potentiometer
* LED
* Jumper Wires

## Topics Covered

* ESP32
* MicroPython
* GPIO Programming
* Analog-to-Digital Converter (ADC)
* Pulse Width Modulation (PWM)
* LED Brightness Control

## Files

* main.py
* diagram.json
* circuit.png
* serial-monitor.png

## Description

This project demonstrates LED brightness control using an ESP32 and a potentiometer. The ESP32 continuously reads the analog voltage from the potentiometer through its ADC, converts it into a digital value, and adjusts the LED brightness using PWM. The corresponding ADC readings and brightness values are displayed in the Serial Monitor, illustrating the relationship between analog input and PWM output. This project provides practical experience with analog signal acquisition, PWM-based output control, and embedded systems programming using MicroPython.
