# Bipolar Stepper Motor Control using ESP32

## Objective
To control a bipolar stepper motor using an ESP32 and an A4988 stepper motor driver with MicroPython.

## Components
- ESP32
- Bipolar Stepper Motor
- A4988 Stepper Motor Driver
- External Power Supply
- Jumper Wires

## Topics Covered
- ESP32
- MicroPython
- GPIO Programming
- Stepper Motor Control

## Files
- main.py
- circuit.png
- diagram.json 

## Description
This project demonstrates how to interface an ESP32 with an A4988 stepper motor driver to control the rotation of a bipolar stepper motor. The ESP32 generates STEP and DIR signals to control the motor's speed and direction, providing precise and reliable motion control for embedded system applications.
